# 因子定义详解

本文档详细说明情绪指数体系中14个核心因子的构造逻辑、原始指标含义和历史百分位解读。

---

## 核心设计原则

所有因子统一采用 **180日滚动百分位化**（取值0–100），将不同量纲的原始指标转换为可比的历史位置分数。百分位越高，说明该指标当前数值在历史上越高。

---

## 各因子详解

---

### 1. obv_factor（能量潮因子）

**原始指标：** OBV（On-Balance Volume）

**构造步骤：**
```
balance_vol = (收盘价_t - 收盘价_{t-1}) × 成交量_t
obv = Σ balance_vol（累加）
obv_change = obv_t - obv_{t-90}      # 90日变化量捕捉中期趋势
obv_factor = 180日百分位化(obv_change)
```

**深入理解：** OBV把每一天的成交量前乘以价格涨跌幅，相当于对每天的"净买入量"做累加。OBV上升说明最近90天资金总体净流入，obv_change为正则主力在加仓，为负则在撤退。这是一个纯粹从资金流角度看问题的指标，不受指数绝对涨跌影响——指数可以横盘，但只要成交活跃、价格上涨，OBV就会上升。

**过热信号（≥80）：** 资金持续净流入累积，上涨动量充沛。但要注意：高位钝化后第一次下降往往是预警（主力可能已悄悄出货）。

**过冷信号（≤20）：** 资金持续净流出，主力撤退明显。历史上每次OBV触及极低值，往往对应重要底部。

---

### 2. mfi_factor（资金流量因子）

**原始指标：** MFI（Money Flow Index）

**构造步骤：**
```
典型价格 Typ = (最高价 + 最低价 + 收盘价) / 3
资金流量 MF = Typ × 成交量
PMF = Σ(上涨日的MF)，rolling(30日)
NMF = Σ(下跌日的MF)，rolling(30日)
MFI = 100 - 100 / (PMF/NMF + 1)         # 类似RSI，但加了成交量权重
mfi_factor = 180日百分位化(MFI)
```

**深入理解：** MFI是RSI的"加量版"。RSI只看价格，MFI把成交量也加权进去了。PMF大说明价格上涨那天成交量也大，是真正的"量价齐升"；NMF大说明价格下跌那天成交也大，是"放量杀跌"。MFI高于80表示最近30个交易日资金流入强度处于历史极值——这是一个短期过热预警，因为极度流入后往往伴随获利了结。

**过热信号（≥80）：** 买盘极度旺盛，短期可能过热，获利回吐压力大。

**过冷信号（≤20）：** 卖压充分释放，往往对应阶段性底部。

---

### 3. leverage_factor（融资杠杆因子）

**原始指标：** 融资买入金额

**构造步骤：**
```
融资买入金额_5日均值 → rolling(5)
leverage_factor = 180日百分位化(融资买入金额_5日均值)
```

**深入理解：** 杠杆型投资者是市场里最敏感的一群人——他们借来的钱有成本，必须快速获利了结。融资余额高说明杠杆资金在积极做多，但同时也意味着一旦市场回调，强制平仓的踩踏会更剧烈。这个因子高的时候往往是市场最乐观的时刻，但也是最脆弱的时刻。

**过热信号（≥80）：** 杠杆资金大规模入场，市场亢奋，但平仓踩踏风险在累积。

**过冷信号（≤20）：** 融资客大规模撤退，情绪极度悲观，往往对应底部。

---

### 4. pcr_factor（期权多空因子）

**原始指标：** 期权持仓量 PCR（Put/Call Ratio）

**构造步骤：**
```
PCR = 认沽期权持仓量 / 认购期权持仓量
pcr_factor = 180日百分位化(PCR)
```

**深入理解：** PCR这个指标很反直觉——PCR高（即看跌期权持仓多于看涨）反而往往是市场短期高点。为什么？因为大量持有看跌期权的往往是对冲基金和机构，他们不是在赌下跌，而是在"买保险"。当他们普遍感到需要买保险时，往往是市场最脆弱的时候。反过来，PCR极低说明市场过于乐观，对冲需求几乎不存在，这种时候往往伴随顶部。

**过热信号（≥80）：** 认沽期权远超看涨，对冲需求激增，市场谨慎，是短期局部高点的预警。

**过冷信号（≤20）：** 看涨远超看跌，市场过度乐观，反转风险累积。

---

### 5. turnover_amount_factor（流动活性因子）

**原始指标：** 全市场成交金额 + 换手率

**构造步骤：**
```
成交金额_5日均值 → 标准化 → tot_amount_factor
换手率_10日均值 → 标准化 → turnover_rate_factor
turnover_amount_factor = (tot_amount_factor + turnover_rate_factor) / 2
（两个标准化指标各占一半）
```

**深入理解：** 这个因子本质上是市场情绪的"温度计"。高换手+高成交说明市场参与者情绪高昂、交易频繁，这在牛市中是正常的，但在牛转熊的关键节点往往也是最后的加速冲顶。极低的换手则说明市场冷清到冰点——历史上这种时候往往是先于大盘见底的信号，因为冷清到极点后只要有一点增量资金，就容易产生反弹。

**过热信号（≥80）：** 投机情绪亢奋，过度拥挤，一旦反转抛压大。

**过冷信号（≤20）：** 市场冷清，往往酝酿底部反转。

---

### 6. ar_factor（人气因子）

**原始指标：** AR（Activative Ratio）

**构造步骤：**
```
AR = Σ(最高价 - 开盘价) / Σ(开盘价 - 最低价)，rolling(60日)
ar_factor = 180日百分位化(AR)
```

**深入理解：** AR的核心看的是"每天收盘价落在日内区间的相对位置"。AR高，说明这些天经常收在日内高位——买家愿意在高位接单，市场人气强；AR低，说明经常收在低位——买家不愿意在高位接单，哪怕指数没怎么跌，每天都是低收盘。这是一种细粒度的市场人气观察，不受大盘绝对涨跌影响。如果指数横了30天但AR持续走低，说明这30天里买家的意愿在系统性衰退。

**过热信号（≥80）：** 持续收于日内高位，买方力量极强，狂热追涨，短期可能过热。

**过冷信号（≤20）：** 持续收于日内低位，卖方主导，人气涣散，往往出现在下跌尾声或横盘磨底阶段。

---

### 7. br_factor（多空买卖意愿因子）

**原始指标：** BR（Bulls/Bears Ratio）

**构造步骤：**
```
BR = Σ(最高价 - 昨收) / Σ(昨收 - 最低价)，rolling(60日)
br_factor = 180日百分位化(BR)
```

**深入理解：** BR与AR的核心区别在于基准点——AR的分子是"开盘价相对最低价的位置"，BR的分子是"最高价相对昨收的位置"。BR衡量的是持仓者在当前价位上还愿不愿意拿着——BR高说明在当前这个位置，买家仍然愿意以高于昨天的价格买入（因为分子是最高价-昨收，差值为正说明最高价高于昨天收盘）；BR低说明持仓者已经不太愿意承受波动，稍有风吹草动就想跑。结合AR一起看能更好地判断市场结构：AR高+BR高=多方人气和信心都强；AR低+BR低=人气信心双弱。

**过热信号（≥80）：** 高位仍强势承接，信心强，但警惕"价格涨但BR阶梯式下降"的顶背离。

**过冷信号（≤20）：** 恐慌抛压充分释放，情绪修复窗口打开。

---

### 8. emascore_long_factor（均线多头因子）

**原始指标：** 多周期均线位置与方向

**构造步骤：**
```
signal0: 收盘价 > MA30 → 0.1分
signal1: MA5 > MA20 → 0.1分
signal2: MA10 > MA20 → 0.1分
signal3: MA10 > MA30 → 0.1分
signal4: MA10 > MA60 → 0.1分
signal5: MA10 > MA120 → 0.1分
signal6: MA20 > MA60 → 0.1分
signal7: MA20向上趋势 → 0.1分
signal8: MA60向上趋势 → 0.1分
signal9: 收盘价 > SAR → 0.1分

jc_mkt_trends_long_term = (signal3+signal4+signal5+signal6+signal8) × 100
emascore_long_factor = 上述长期信号的30日滚动均值
```

**深入理解：** 这个因子本质上是在问：市场的中长期均线有没有形成多头排列？它综合评估了MA10/MA20/MA60/MA120的位置关系和方向，9个信号各0.1分，满分0.9分。因子值反映的是"均线多头排列的健康程度"——不是单纯看价格是否在均线上方，而是同时看各条均线自身的方向是否向上。比如价格可以在MA10上方但MA60还在下降，这时signal4=0.1但signal8=0，因子值偏低。

**过热信号（≥80）：** 均线多头排列完整，趋势向上，做多信号。

**过冷信号（≤20）：** 均线空头或混乱排列，趋势向下，不宜做多。

---

### 9. signal_macd_factor（MACD动量因子）

**原始指标：** MACD（Moving Average Convergence Divergence）

**构造步骤：**
```
EMA1 = 收盘价的10日指数移动平均
EMA2 = 收盘价的25日指数移动平均
DIFF = EMA1 - EMA2
DEM = DIFF的9日移动平均
MACD = DIFF - DEM

signal7_macd: (MACD今日 - MACD昨日 > 0) → 1 : 0
signal8_macd: (MACD今日-MACD昨日 > 0 且 DIFF > 0) → 1 : 0
signal9_macd: (DIFF > 0) → 1 : 0

signal8_macd_sum = Σ signal8_macd，rolling(20日)
signal9_macd_sum = Σ signal9_macd，rolling(20日)
signal_macd_factor = 180日百分位化((signal8_macd_sum + signal9_macd_sum)/2)
signal_macd_factor = 上述的20日滚动均值
```

**深入理解：** MACD是技术分析里最经典的趋势指标之一。这个因子的设计重点在于两个维度：signal8看MACD是否在上升（动量方向），signal9看DIFF是否在零轴上方（趋势方向）。signal8_macd_sum+signal9_macd_sum再rolling(20日)是为了平滑噪声。简单说：因子高=MACD多头信号持续且DIFF在零轴上方=上涨趋势健康；因子低=MACD空头信号持续=趋势向下。值得特别注意的是MACD与价格的背离——价格创新低但MACD没有创新低，往往是历史性底部信号。

**过热信号（≥80）：** MACD多头信号强，趋势向上延续性好，但高位钝化要警惕。

**过冷信号（≤20）：** MACD空头主导，趋势向下，可能酝酿底部背离。

---

### 10. highlow_factor（高低价动量因子）

**原始指标：** 最高价/最低价的10日斜率比较

**构造步骤：**
```
涨市信号：收盘价上涨 且 最高价10日斜率 > 最低价10日斜率 → 1分
跌市信号：收盘价下跌 且 最低价斜率 < 最高价斜率（底部有支撑）→ 1分
highlow_signal = Σ(信号)，rolling(20日)
highlow_factor = 180日百分位化(highlow_signal)
```

**深入理解：** 这个因子解决的是一个很实际的问题：上涨的时候，是"真的强"（最高价持续创新高）还是"虚的强"（只有收盘价涨，最高价其实在衰减）？highlow_factor就是来判断这个的。涨势中最高价斜率持续领先最低价斜率=真正的买方主导；跌势中最低价斜率相对收缩（没有创新低）=底部有资金承接。该因子变化较快，在横盘震荡行情中可能反复波动，不宜作为单一趋势信号。

**过热信号（≥80）：** 连续20日高低价斜率显示买方主导，动量极强，但变化快需警惕。

**过冷信号（≤20）：** 连续20日呈现弱势或反转特征，往往对应阶段性底部。

---

### 11. RSI_factor（相对强弱因子）

**原始指标：** RSI（Relative Strength Index）

**构造步骤：**
```
RSI1 = 12日周期 RSI
RSI2 = 25日周期 RSI
RSI3 = 40日周期 RSI
signal10_RSI = Σ RSI1，rolling(5日)
RSI_factor = 180日百分位化(signal10_RSI)
```

**深入理解：** RSI是判断超买超卖最经典的指标。这个因子的特别之处在于它不是用RSI的绝对值，而是用RSI1的5日累加值来做百分位化——这样做的好处是平滑了单周期的假信号，同时捕捉了RSI的绝对水平。多周期RSI（12/25/40日）融合减少了一个周期参数变化带来的噪声。值得注意的是：RSI>70是教科书式的超买信号，但在一个强势趋势里RSI可以长时间维持高位钝化不跌；RSI<30是超卖信号，但弱势趋势里可以长时间维持低位。

**过热信号（≥80）：** RSI处于历史高位，短期回调压力大。

**过冷信号（≤20）：** RSI处于历史低位，超卖明显，布局机会区。

---

### 12. daily_return_factor（日收益率因子）

**原始指标：** 日收益率的30日移动均值

**构造步骤：**
```
日收益率 = (收盘价 - 昨收盘价) / 昨收盘价
日收益率_30日均值
daily_return_factor = 180日百分位化(日收益率_30日均值)
```

**深入理解：** 这是最直接反映短期市场表现的因子——最近30个交易日的平均日收益率在历史上排什么位置。本质上是一个趋势追踪指标，和综合情绪指数高度相关但更纯粹地反映"价格动量"。值得注意的是：这个因子对市场短期涨跌的反应最直接，所以变化也最灵敏，在情绪指数出现拐点之前它往往最先反应。

**过热信号（≥80）：** 短期累计涨幅历史高位，获利回吐压力大。

**过冷信号（≤20）：** 短期累计跌幅历史低位，超跌反弹概率大。

---

### 13. up_number_rate_factor（市场广度因子）

**原始指标：** 全市场上涨股票比例

**构造步骤：**
```
上涨比例 = 上涨股票数 / (上涨 + 下跌股票数)
上涨比例_20日均值 → rolling(20)
up_number_rate_factor = 180日百分位化(上涨比例_20日均值)
```

**深入理解：** 这个因子解决的是"指数有没有失真"的问题。大盘指数可能因为少数大盘股拉升而看起来还行，但大多数股票已经在跌——这时候指数是失真的。up_number_rate_factor就是在衡量这种广度。因子高=全市场普涨，人气蔓延健康；因子低=全市场普跌，恐慌主导。更重要的是背离信号：如果指数在走强但up_number_rate在走弱（分化），说明上涨全靠少数权重股硬撑，大多数股票已经在撤退，这是预警信号。

**过热信号（≥80）：** 绝大多数个股上涨，情绪高度一致，注意踩踏风险。

**过冷信号（≤20）：** 绝大多数个股下跌，恐慌主导，冰点后往往反弹。

---

### 14. equity_bond_effective_factor（广义拥挤度因子）

**原始指标：** 沪深300 PE倒数 + 全市场成交金额

**构造步骤：**
```
E/P = 1 / 沪深300_PE（赔率）
E/P_标准化 = 180日百分位化(E/P)
成交金额_标准化 = 180日百分位化(成交金额)
gap = E/P_标准化 - 成交金额_标准化
gap_margin = gap_5日均值 - gap_60日均值
equity_bond_effective_factor = 100 - 180日百分位化(gap_margin)
```

**深入理解：** 这是整个体系里设计最精巧、也是最有预警价值的因子。它的核心逻辑是：**赔率低（股票贵）+ 成交活跃（市场热）= 最危险的组合**。分子E/P标准化衡量的是沪深300相对债券的吸引力（E/P高=股票比债券划算=赔率高）；成交金额标准化衡量的是市场热度（成交大=热）。gap_margin的5日均值-60日均值衡量的是边际变化——最近5天的拥挤度变化相对长期是扩张还是收缩。最后取100减去百分位，所以gap_margin越低（越拥挤），因子读数越高。这解释了为什么因子高（≥80）是最危险的信号：赔率低+成交热+还在加速涌入。

**过热信号（≥80，即gap_margin极低）：** 极度拥挤+低赔率，历史回测显示这是最重要的反转预警信号之一。

**过冷信号（≤20，即gap_margin极高）：** 低拥挤+高赔率，历史上较好的布局区间。

---

## 重要：指标之间的配合使用

**趋势行情确认：** emascore_long_factor + signal_macd_factor + up_number_rate_factor 三者同向 = 趋势健康

**预警类信号：** equity_bond_effective_factor ≥ 80 + pcr_factor ≥ 80 = 高置信度风险预警

**底部信号：** mfi_factor ≤ 20 + RSI_factor ≤ 20 + up_number_rate_factor ≤ 20 = 多指标共振底部

**背离预警（最重要）：** 价格创新高但equity_bond_effective_factor下降，或价格创新低但RSI未创新低 → 历史胜率极高的反转信号
