# index-sentiment-analyzer

OpenClaw skill for analyzing Chinese equity market sentiment via the CSI 300 ETF sentiment index.

## What it does

- 14-factor sentiment analysis (0–100 percentile scores)
- Composite sentiment index with tiered readings
- Early warning signal detection
- Factor deep-dives and trend interpretation

## Usage

Triggered automatically when asking about:
- A股 / Chinese stock market sentiment
- 沪深300 / CSI 300 sentiment or index analysis
- Market mood, buyer/seller sentiment indicators

## Factors

obv, mfi, leverage, pcr, turnover, ar, br, rsi, ema, macd, highlow, return, breadth, crowding

## Data

Live data from Feishu spreadsheet on every query. No cache needed.

## Disclaimer

For research and education only. Not investment advice.
