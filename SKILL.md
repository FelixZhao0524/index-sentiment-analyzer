## 各节填充规则

### ▎一 — 综合情绪指数

**档位表：**

| 读数区间 | 档位 | 市场环境 |
|---|---|---|
| 95–100 | 🔴 历史极值 | 极端亢奋 |
| 80–94 | 🟠 过度乐观 | 严重过热 |
| 65–79 | 🟡 乐观偏热 | 偏热运行 |
| 50–64 | 🟢 中性偏暖 | 正常偏多 |
| 35–49 | 🟢 中性偏冷 | 正常偏弱 |
| 20–34 | 🔵 悲观偏冷 | 明显冷却 |
| 5–19 | 🔵 过度悲观 | 接近冰点 |
| 0–4 | 🔵 冰点 | 历史极冷 |

**市场环境解读：**
- ≥80：市场情绪极度亢奋，杠杆资金、融资余额、成交量等指标均处历史高位，须警惕均值回归
- 65–79：情绪偏热但未至极端，多头资金仍为主导，注意边际拐点
- 50–64：情绪中性偏暖，多空力量均衡，趋势方向不明
- 35–49：情绪中性偏差，资金入场动力不足，以防御为主
- ≤34：情绪处于历史偏低水平，往往对应布局机会区

**较昨日变化：**
- 上升>5：情绪大幅回暖，资金快速涌入，做多情绪显著升温
- 上升1–5：情绪稳中向好，做多力量温和增强
- ±1以内：情绪基本平稳，无明显方向偏好
- 下降1–5：情绪有所回落，做多力量略有消退
- 下降>5：情绪快速降温，出现明显恐慌或获利了结迹象

### ▎二 — 趋势与动能

**5日斜率解读：**

| 斜率区间 | 解读 |
|---|---|
| > +2.5 | 情绪强势升温，短线过热风险大 |
| +1.5 ~ +2.5 | 情绪明显升温，趋势向上概率大 |
| +0.5 ~ +1.5 | 情绪温和改善，方向偏多但力度不强 |
| -0.5 ~ +0.5 | 情绪平稳，无明确方向 |
| -1.5 ~ -0.5 | 情绪温和降温，做多力量消退 |
| -2.5 ~ -1.5 | 情绪明显降温，空头力量主导 |
| < -2.5 | 情绪快速退潮，恐慌或顶部风险 |

**EMA5-EMA20差值解读：**

| EMA差值 | 解读 |
|---|---|
| > +5 | EMA5上穿EMA20形成金叉，中期趋势由弱转强 |
| +2 ~ +5 | EMA5位于EMA20上方且张口扩大，多头格局健康 |
| -2 ~ +2 | EMA张口极小，中期方向不明，等待突破 |
| -5 ~ -2 | EMA5位于EMA20下方且张口扩大，空头格局延续 |
| < -5 | EMA5下穿EMA20形成死叉，中期趋势由强转弱 |

### ▎三 — 预警信号

**触发条件：** `sentiment_now >= 99` AND 次日下降（拐点出现）

**触发时：**
```
预警触发！情绪读数已达历史极值（{sentiment_now:.0f}）并出现拐点。
历史7次触发记录中6次在随后约20个交易日内出现超过10%回撤。
平均最大回撤{-13.77%}，单次最大{-32.46%}（2017年末）。
建议高度重视风险。
```

**未触发时：**
- sentiment_now < 90：读数尚未进入过热区间（<90），预警条件未满足
- sentiment_now >= 90 且仍在上升：读数处于历史高位但尚未出现拐点
- sentiment_now >= 99 且未下降：需等待下降拐点确认

### ▎四 — 重点因子追踪（核心章节）

**写作要求：** 选取3–5个最有解释力的因子，结合通俗含义 + 当前数值 + 市场含义，写出**有判断的段落**，而不是罗列数字。

**因子选取优先级：**
1. 所有 ≥80 或 ≤20 的极值因子（必选）
2. 综合情绪指数读数对应组内最有代表性的因子
3. 与近期变化方向矛盾或有特殊意义的因子

**写法示范（ar_factor = 10 为例）：**
> 人气因子AR处于历史冰点附近（10），表示沪深300收盘价近期持续落在日内低点区间——即使指数没怎么跌，买家也不愿意在日内高位接手，说明短线做多人气涣散，市场信心不足。这类信号单独出现未必致命，但往往出现在下跌趋势的尾声或横盘磨底阶段。

### ▎五 — 综合研判

**模板（但不僵化）：**
```
综合读数{sentiment_now:.0f}（{档位}），{一句话定性}。
{趋势方向}，{动能状态}。
{最大矛盾或亮点}。
{总体评价}。
```

**档位对应的机会/风险定性：**
- ≥80：回调风险大于上涨空间，高位追涨风险极大，以防御减仓为主
- 65–79：可继续持有，但不宜新开重仓，密切观察边际拐点
- 50–64：方向不明，保持现有仓位，等待明确信号
- 35–49：控制仓位为主，部分因子已处低位，关注修复机会
- ≤34：历史低位往往对应重要底部区域，可逐步关注左侧布局机会

## 因子通俗解读（参考手册）

以下为14个因子的金融含义，AI在分析时应自由引用，不必按固定顺序输出。

| 因子 | 通俗解释 | 高值（≥80）含义 | 低值（≤20）含义 |
|------|----------|----------------|----------------|
| **obv_factor** | 资金流向——钱往哪里流 | 主力资金持续净流入，中期做多信号 | 资金持续净流出，主力撤退 |
| **mfi_factor** | 资金流入强度——买盘有多旺 | MFI资金流量达极值，买盘极度旺盛，短期可能过热 | 卖压充分释放，往往对应阶段性底部 |
| **leverage_factor** | 融资热度——杠杆资金有多热 | 融资客大规模入场，市场亢奋，警惕平仓踩踏 | 融资客大规模撤退，情绪极度悲观，底部信号 |
| **pcr_factor** | 对冲需求——恐慌有多强 | 期权看跌持仓远超看涨，市场谨慎，反向信号 | 期权看涨远超看跌，市场过度乐观，反转风险累积 |
| **turnover_amount_factor** | 换手热度——市场有多活跃 | 成交极度活跃，投机情绪亢奋 | 成交极度萎缩，市场冷清，往往是底部 |
| **ar_factor** | 人气因子——收盘在日内什么位置 | 持续收于日内高点，买方力量极强，狂热追涨 | 持续收于日内低点，卖方主导，往往对应恐慌底 |
| **br_factor** | 买卖意愿——持仓信心强弱 | 资金高位仍强势承接，信心强，警惕动能衰减 | 恐慌抛压充分释放，情绪修复窗口打开 |
| **RSI_factor** | 超买超卖——涨过头还是跌过头 | RSI处历史高位，超买明显，短期回调压力大 | RSI处历史低位，超卖明显，布局机会区 |
| **daily_return_factor** | 短期累计收益 | 短期涨幅达历史高位，获利回吐压力大 | 短期跌幅达历史低位，超跌反弹概率大 |
| **equity_bond_effective_factor** | 拥挤度——市场有多挤 | 极度拥挤（数值极小=最拥挤），最危险的反转信号 | 低拥挤+高赔率，历史较好布局区间 |
| **emascore_long_factor** | 均线多头排列强度 | 均线完美多头，趋势向上，做多信号 | 均线空头排列，趋势向下 |
| **signal_macd_factor** | MACD动量方向 | MACD多头信号强，上涨趋势健康 | MACD空头主导，趋势向下 |
| **highlow_factor** | 高低价动量——谁在主导 | 高价斜率持续领先低价，买方主导 | 低价斜率扩大或收缩，动量衰竭，趋势可能反转 |
| **up_number_rate_factor** | 市场广度——涨跌家数比 | 全市场普涨，情绪高度一致，注意踩踏风险 | 全市场普跌，恐慌主导，冰点后往往反弹 |
